Gameplay:
	Basically, player will try to defeat as much pokemon as possible. There 10% player will encounter a wild Pokemon whenever the player walks in the Bush area. Player has 3 choices while encountering a Pokemon: Run, Catch and Fight. 
		Run: Player will run away from encoutering Pokemon.
		Catch: Player will try to catch encountering Pokemon. The less HP encoutering Pokemon has, the more percentages that 
			Player is able to catch it. After catching, Player can switch Player's Pokemon to catched Pokemon.
		Fight: Player's pokemon will attack encoutering Pokemon and reversed. The higher level Pokemon will go first.
		
		
	Player is able to choose Trainer and starter Pokemon before get in the main game.
	There is an intruction screen which is known as Lecture Screen before player jump in main Game.
